# Current

## Operating state on 2026-04-25

| Layer | Current reality |
|---|---|
| Lead acquisition | Instagram hashtag and follower scraping, manual DM opening |
| Opt-in | Free fitness group plus personalized plan request |
| Data capture | Google Form plus email collection |
| CRM | Teamzy manages warmth and follow-up from L2 onward |
| Customer conversion | Warm L2 leads become product customers and enter a VIP chat |
| Multiplication | Loyal VIPs can become ambassadors |
| Content | Reels and inspirational shorts exist, but not as a canonical funnel layer |

## Primary current constraint

Rob explicitly identifies L1 prospecting as the place he spends the most time. The strategy docs agree: the funnel works, but the top of the funnel is manually expensive.

## Current strengths

- Real founder-led sales process
- Personalized free plan offer
- CRM discipline after the initial conversion
- Clear VIP and ambassador progression
- Existing content fragments that can be systematized

## Current weaknesses

- No formal content inventory
- No shared naming scheme for reels, content packs, or pillar categories
- No warm-first routing rule visible in the current process
- Limited public proof layer relative to the amount of manual outreach
