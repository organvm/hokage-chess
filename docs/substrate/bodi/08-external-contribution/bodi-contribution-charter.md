# BODI Contribution Charter

## Valuable contribution types

- Rob sends reel links, handles, screenshots, and examples of what he already posts.
- Collaborators classify assets by topic, platform, and funnel role.
- Researchers add actor cards that extract mechanisms rather than admiration.
- Operators add measured process improvements that reduce manual drag or improve retention.

## Intake rules

1. Every new asset should be attached to a concrete file in this tree.
2. Raw links belong in the stratum they inform, not in scattered notes.
3. Claims about performance or external figures should identify their source.
4. Work that changes the funnel should explain which stage it affects and what pressure it removes.

## Standard for acceptance

A contribution is useful when it makes one of these easier:

- attracting warmer leads
- converting L2 without extra founder hours
- retaining VIPs with real value
- multiplying ambassadors with less improvisation
- understanding which outside systems are worth stealing from
