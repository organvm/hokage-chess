# BODI Substrate

This directory is the first executable instance of the domain substrate plan for Rob Bonavoglia's BODI business. It translates the existing strategy documents into eight stable strata so future research, comparisons, and interventions land in a predictable place.

## Canonical local sources

- `../../business/2026-04-25-bodi-funnel-with-premium-content-architecture.md`
- `../../business/2026-04-25-cross-pollination-master-diagnosis.md`
- `../../business/2026-04-25-rob-call-transcript-source.md`

## Current status

- The funnel is already real and operational.
- The premium content layer exists in fragments but is not yet installed as a system.
- The cross-pollination diagnosis with Hokage Chess is the current highest-leverage strategic insight.
- The constellation method has five named seed figures and room to expand.

## Strata

1. `01-ontology/` - entities, relations, tools, claims
2. `02-lineage/` - historical anchors, current state, future leads
3. `03-constellation/` - people and sibling systems to study
4. `04-gap-map/` - what is missing or underbuilt
5. `05-agent-fleet/` - repeatable operator roles
6. `06-production-stack/` - tools, channels, rhythms, assets
7. `07-internal-magnet/` - what BODI should absorb, improve, or reject
8. `08-external-contribution/` - what collaborators can add and how it gets normalized
