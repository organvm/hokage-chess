# Actor Card: Autumn Calabrese

**Domain:** BODi Core
**Confidence:** 5/5 (Globally Recognized)

## Profile
Creator of *21 Day Fix*, the program that fused workouts with a structured nutrition system and made eating, not just exercising, the core of the offer.

## Key Contributions
- **Portion-Control Containers:** Codified a color-coded container system that simplified nutrition into a repeatable, beginner-proof ruleset.
- **Nutrition-First Funnel:** Made diet the primary value driver, opening a recurring product and upsell path beyond workout content.
- **21-Day Commitment Frame:** Used a short, defined window to lower the barrier to starting and to set a clear finish line.

## Relevance to BODI
Models nutrition-first funnel discipline: a simplified, systematized rule layer that beginners can follow without judgment. Rob extracts how a constrained, easy-entry system drives both completion and recurring revenue.
